# apollo-graphql-codgen-fragment-colocation

## Usage

### Run web app

```
npm run dev
```

### Run GraphQL server

```
npm run mock
```

### Generate GraphQL code using GraphQL Codegen

See https://www.the-guild.dev/graphql/codegen/docs/guides/react-vue


```
npm run codegen

or

npm run codgen:watch
```
